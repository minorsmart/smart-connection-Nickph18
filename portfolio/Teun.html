Introductie

Algemene bevinding tijdens de Minor Smart Industry


Als ondernemende bedrijfskundige student die de minor Smart Industry heeft gevolgd, kijk ik vol trots terug op mijn ervaring. Deze minor heeft mijn kennis en vaardigheden naar een hoger niveau getild en heeft mij voorbereid op de uitdagingen van de moderne industrie. Mijn kennis is verbreed en ik heb het gevoel dat ik mijn leerdoelen heb mogen behalen.

Ik heb geleerd om geavanceerde technologieën zoals het Internet of Things en data-analyse toe te passen in een business perspectief. Door de verschillende projecten en samenwerking met bedrijven, heb ik veel ervaring opgedaan en heb ik meer geleerd over hoe bedrijven technologie kunnen toepassen. Deze minor heeft mijn kennis verbreed en voorbereid op de technologieën van de toekomst. Met name hoe je deze technologie kan toepassen binnen het bedrijf vond ik enorm interessant. Ik heb door de minor zeker een stap voor op andere studenten van mijn opleiding. Hierdoor kijk ik met een goed gevoel terug op de minor.

Portfolio

HTML & CSS

Het bouwen van ons portfolio is compleet in HTML gedaan. Ik heb op mijn opleiding en daarbuiten nog nooit gebruik gemaakt van HTML. Voor mij was dit dus compleet nieuw. De eerste les van Witek maakte voor mij de basis duidelijk en ik kwam er eigenlijk achter dat ik gewoon moest gaan proberen. De basis van de site en de CSS heeft voornamelijk Nick verzorgd. Hij heeft ons uitgelegd dat bij HTML de algemene opmaak te vinden is in de CSS mapjes.

Mijn eigen pagina heb ik zonder hulp weten te customizen. Ik heb op eigen initiatief mijn eigen CSS stylesheet toegevoegd, de originele gekopieerd en hierin aanpassingen gedaan die ik zelf wilde doen. Hierdoor kon ik mijn eigen pagina laten onderscheiden van de andere pagina's op onze site.
